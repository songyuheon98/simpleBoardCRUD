# Simple Board 

---
## 토이 프로젝트 배경
---
최근 입사한 회사에서 과제를 받게 되었습니다. 이 과제는 CRUD(생성, 읽기, 업데이트, 삭제) 기능을 구현하는 것입니다. 이번 기회에 그동안 이름만 들어봤지 실제로 사용해보지 못했던 MyBatis와 전자정부프레임워크, JSP를 활용해 CRUD 게시판을 직접 만들어보고자 합니다.

---
## 목표
---
### CRUD를 4가지 방법으로 구현하는 것이 목표입니다. 
---
1. MyBatis
2. JDBC
3. JPA
4. QueryDSL
